Regeln

Denk wie bei einer Hunt

Nicht alles ist direkt sichtbar. Achte auf Format, Wortwahl, Buttons, Quelltext, auffällige Begriffe, versteckte Hinweise, Wikipedia-artige Antworten und Kombinationen aus früheren Lösungen.

  • Groß-/Kleinschreibung ist egal.
  • Bei manchen Rätseln ist ein Format vorgegeben.
  • Einzelne Hinweise sind absichtlich im Code versteckt.
  • Nicht jedes Rätsel ist nur eine Verschlüsselung.
